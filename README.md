# Final Offer UI

Multi-page real estate platform UI built with HTML/CSS/JS.

## Live Preview
[View live](https://astghika-creator.github.io/finaloffer-ui)

## Pages
- **My Feed** - Activity stream with filter tabs
- **My Listings** - Property cards with status badges
- **My Offers** - Offer pipeline with stats
- **Channels** - Messaging hub
- **Leads** - Hot/warm/cold pipeline
- **Contacts** - Contact directory
- **Home Tours** - Tour scheduling
- **Account** - Profile settings

## Design Tokens
| Token | Value | Usage |
|-------|-------|-------|
| `--ink` | `#02121a` | Primary text |
| `--muted` | `#6a859d` | Secondary text |
| `--border` | `#f0f2f7` | Dividers |
| `--bg-warm` | `#f9fafc` | Sidebar bg |
| `--red` | `#e0362c` | Badges, dots |
| `--navy` | `#0e2340` | Brand badge |

## Responsive
- Desktop >= 1024px: 3-column layout
- Tablet 768-1023px: Icon sidebar
- Mobile <= 767px: Bottom tab bar
