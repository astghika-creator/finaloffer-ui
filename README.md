# Final Offer UI

Multi-page real estate platform UI built with HTML/CSS/JS.

## Live Preview
[View live](https://astghika-creator.github.io/finaloffer-ui)

## Pages
- **My Feed** - Activity stream with filter tabs
- **My Listings** - Property cards with status badges
- **My Offers** - Offer pipeline with stats table
- **Channels** - Messaging hub with unread badges
- **Leads** - Hot/warm/cold pipeline
- **Contacts** - Contact directory
- **Home Tours** - Tour scheduling
- **Account** - Profile settings

## Design Tokens
| Token | Value | Usage |
|-------|-------|-------|
| `--ink` | `#02121a` | Primary text, active states |
| `--muted` | `#6a859d` | Secondary text, inactive nav |
| `--border` | `#f0f2f7` | Dividers, pill borders |
| `--bg-warm` | `#f9fafc` | Sidebar, page shell |
| `--red` | `#e0362c` | Unread dots, badges |
| `--green` | `#3ab44a` | Success states |
| `--navy` | `#0e2340` | Brand badge background |

## Responsive Breakpoints
- **Desktop** >= 1024px - Full 3-column layout with right panel
- **Tablet** 768-1023px - Icon-only sidebar, no right panel
- **Mobile** <= 767px - Bottom tab bar, stacked single-column

## Files
- `index.html` - Full multi-page app (always latest)
- `finaloffer-master.html` - Feed-only baseline reference
